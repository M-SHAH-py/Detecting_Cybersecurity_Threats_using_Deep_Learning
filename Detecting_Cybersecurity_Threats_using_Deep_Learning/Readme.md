# Detecting Cybersecurity Threats using Deep Learning

![Cybersecurity](cyber_photo.jpg)

## Overview

This project implements a Deep Learning model for cybersecurity threat detection using PyTorch. The model is trained on labelled cybersecurity event data to classify whether a process or activity is malicious or benign.

The notebook demonstrates the complete machine learning workflow, including data preprocessing, feature scaling, model training, validation, testing, and performance evaluation.

---

## Features

- Data preprocessing using Pandas
- Feature normalization with Scikit-learn
- Deep Learning model using PyTorch
- Binary classification
- Training, validation and testing pipeline
- Performance evaluation
