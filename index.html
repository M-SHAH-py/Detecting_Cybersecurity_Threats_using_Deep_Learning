Detecting Cybersecurity Threats using Deep Learning

Deep Learning • PyTorch • Machine Learning • Cybersecurity

Project Overview

This project develops a Deep Learning model capable of identifying cybersecurity threats from labelled system activity data. Using PyTorch, the workflow includes preprocessing, feature scaling, model training, validation, and performance evaluation to classify activities as malicious or benign.

Technologies Used

  • Python
  • PyTorch
  • Pandas
  • NumPy
  • Scikit-Learn
  • TorchMetrics
  • Jupyter Notebook

Project Workflow

  1. Load labelled datasets
  2. Clean and preprocess data
  3. Normalize features
  4. Train Deep Learning model
  5. Validate model
  6. Evaluate on test dataset
  7. Analyze classification performance
