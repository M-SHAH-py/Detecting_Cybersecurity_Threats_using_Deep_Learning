# 🛡️ Detecting Cybersecurity Threats using Deep Learning

A Deep Learning-based Cybersecurity Threat Detection system developed using **PyTorch** to classify network traffic and identify malicious activities. This project demonstrates a complete machine learning workflow, from data preprocessing to model training and evaluation on labeled cybersecurity datasets.

## 🌐 Live Project Website

**GitHub Pages:**

## 📌 Project Overview

Cybersecurity has become one of the most critical challenges in today's digital world. Traditional rule-based detection systems often struggle to identify previously unseen attacks. This project applies **Deep Learning** techniques to detect cybersecurity threats by learning complex patterns from labeled network traffic data.

The repository contains the complete workflow, including:

* Data preprocessing
* Feature preparation
* Deep Learning model development
* Model training
* Validation
* Performance evaluation

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/M-SHAH-py/Detecting_Cybersecurity_Threats_using_Deep_Learning.git
```

### Navigate to the Project

```bash
cd Detecting_Cybersecurity_Threats_using_Deep_Learning
```

### Install Required Libraries

```bash
pip install torch pandas numpy scikit-learn torchmetrics notebook
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```text
Cybersecurity_Threat_Detection.ipynb
```
